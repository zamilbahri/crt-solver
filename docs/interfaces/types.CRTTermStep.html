    Interface CRTTermStep

    Represents a single computational step in the CRT algorithm for one equation.

    Contains all intermediate values computed for the equation x ≡ a_i (mod m_i), including the original input, normalization, partial products, and extended GCD coefficients used to derive the modular inverse.
